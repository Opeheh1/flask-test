# Flask Healthcare Application

This is a Python project that develops a data driven survey tool to collect and analyze participants’ financial behavior, it designs an end-to-end system that captures user demographic and financial data, stores it efficiently, and transforms it into actionable insights for business decision-making. It integrates Flask for web development, MongoDB for data storage, and Python for data processing and visualization.

## Features

- Develop a user-friendly web interface using Flask.
- Store collected user data in a MongoDB database
- Create a class (User) to represent and manage user data and export it as a csv file.
- Generate analytical visualizations
- Prepare visual outputs
- Deploy the Flask application on AWS(EC2 Instance)



## Technology

1. Python 3
2. IDE (PyCharm)
3. Library (flask, pymongo)

## Installation

1. Install python 3 on your system

2. Install an IDE (e.g PyCharm)

3. In the provided zip file, copy the folder "Flask_healthcare_application" to your python working directory

4. Open the python file "app.py" and run it, and website url will be generated in the console.

5. Open the jupyter file "visualization.ipynb" and run it, the visualization will be generated.

6. In that zip file provided, there is an already exported visualization for the powerpoint presentation.

7. The public IPv4 address for the deployed flask app is 52.91.217.89
